Eneo - En generell AI-plattform

En skalbar digital infrastruktur och AI-plattform utvecklad utifrån kommunernas behov och med öppenhet och delning som grundprinciper.

Om Eneo

Eneo är en gemensam AI-plattform som möjliggör för kommuner att effektivt utveckla, drifta och skala AI-baserade tjänster. Plattformen erbjuder en gemensam grund för automatisering, analys och innovation inom kommunal verksamhet.

Genom att använda Eneo kan kommunerna dra nytta av modern AI-teknik utan att behöva bygga upp egen kompetens och infrastruktur från grunden. Plattformen är utvecklad med fokus på säkerhet, skalbarhet och användarvänlighet.

Funktioner

  • AI-modeller - Tillgång till förtränade AI-modeller anpassade för kommunal verksamhet
  • Automatisering - Verktyg för att automatisera repetitiva arbetsuppgifter
  • Textanalys - Analysera och sammanfatta dokument, ärenden och feedback
  • Chatbot-ramverk - Bygg AI-drivna chatbotar för invånar- och medarbetartjänster
  • Integrationer - Färdiga kopplingar mot vanliga kommunala system
  • API:er - Öppna gränssnitt för att bygga egna lösningar

Så fungerar det

Eneo levereras som en molnbaserad tjänst som kommunerna får tillgång till via säkra anslutningar. Plattformen kan användas både via ett webbaserat gränssnitt och genom API:er för integration med befintliga system.

Sundsvalls och Ånge kommun ansvarar för drift och vidareutveckling av plattformen. Alla kommuner i Kommuna-samarbetet bidrar med behov och utvecklingsinsatser, och tjänster som skapas kan återanvändas av samtliga parter.

Tekniska krav

  • HTTPS-anslutning
  • Integration via API eller webbgränssnitt
  • Modern webbläsare (Chrome, Firefox, Edge, Safari)
  • Identitetsfederation (Microsoft Entra ID eller ADFS)

Möjliga nyttor

  • Minskad administrativ tid genom automatisering av rutinuppgifter
  • Snabbare handläggning av ärenden med AI-stödd analys
  • Förbättrad tillgänglighet för invånare dygnet runt
  • Delade kostnader mellan kommunerna
  • Ingen egen AI-kompetens krävs för att komma igång
  • Skalbar lösning som växer med behoven
  • Säker hantering av känslig information
  • Återanvändning av lösningar mellan kommuner
