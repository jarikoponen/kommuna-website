Teknisk dokumentation

Teknisk dokumentation

Här samlar vi länkar till de öppna källkodsrepon på GitHub som bygger upp Generisk ärendehantering, samt vilka API:er respektive webb-app integrerar med.

Webb-appen

Webb-apparna ligger i ett gemensamt repo som innehåller flera webb-appar. Följande webb-appar mappar mot Generisk ärendehantering.

github.com/Sundsvallskommun/web-app-draken-public

API:er per webb-app

Under varje webb-app listas de API:er som webb-appen måste integrera med. De flesta API:er finns som öppna repon på Sundsvalls kommuns GitHub och länkas direkt nedan. Vissa API:er (t ex Citizen, ActiveDirectory, Employee, LegalEntity och Estateinfo) har inget publikt repo och listas därför utan länk.
