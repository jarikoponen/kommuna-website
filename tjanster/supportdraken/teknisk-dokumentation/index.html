Teknisk dokumentation

Teknisk dokumentation - Generisk ärendehantering

Här samlar vi länkar till de öppna källkodsrepon på GitHub som tillsammans bygger upp helhetslösningen, samt de API:er som lösningen integrerar med.

Webb-appen

Webb-apparna ligger i ett gemensamt repo som innehåller flera webb-appar. Tillsammans bygger de upp en supportlösning som – tack vare att den är konfigurerbar – kan användas i många olika verksamheter, från kontaktcenter och intern support till lön, pension och rekrytering.

github.com/Sundsvallskommun/web-app-draken-public

API:er som lösningen integrerar med

Helhetslösningen integrerar med ett antal API:er. De flesta finns som öppna repon på Sundsvalls kommuns GitHub och länkas direkt nedan.

  • SupportManagement
  • Templating
  • Party
  • Relations
  • CaseStatus
  • BillingPreprocessor
  • SimulatorServer

API:er utan publikt repo

Följande API:er hanterar kommun-specifik master-data och har därför inget publikt repo. För att få dessa på plats krävs en utvecklingsinsats anpassad efter den egna kommunens källsystem.

  • Citizen
  • ActiveDirectory
  • Employee
  • LegalEntity
  • Estateinfo
